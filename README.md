# Simple Calculator 

## About

This is a simple calculator built with ReactJs. It can perform basic mathematical operations like addition, subtraction, multiplication, and division.

## Technologies Used

- **React** - JavaScript library for building user interfaces
- **useState Hook** - Used to manage the calculator's state (current value, previous value, operation)
